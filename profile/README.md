## 성과
- 장바구니 상시 노출, 즉시주문 기능 등 UX 개선으로 주문 클릭 수를 **7회→3회**로 단축
- POS DB 직접 연동을 통해 현재 테이블 주문만 조회 가능, 평균 응답 속도 **6.2초→0.2초 (96% 개선)**  
- 실시간 슬라이드·품절 반영으로 하루 평균 39회(3회 × 13테이블) 순회 업무 제거
